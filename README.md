## Steps to Run the Project Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/Ahsuu27488/ui-ux-hackathon.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ui-ux-hackathon
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
